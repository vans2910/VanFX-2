# GitHub Pages Public URL

Once your repository is created and pushed, the public link will use this format:

`https://<your-github-username>.github.io/<repo-name>/`

## Example

If your GitHub username is `yourname` and the repository is named `vanfx-trial`, the public URL will be:

`https://yourname.github.io/vanfx-trial/`

## How to use it

1. Create and push your repository to GitHub.
2. Make sure the repo is public.
3. Allow GitHub Pages to publish from the `master` branch.
4. Open the URL using the pattern above.

## Notes

- Replace `<your-github-username>` with your actual GitHub username.
- Replace `<repo-name>` with the exact repository name.
- If you later choose the `main` branch instead of `master`, the URL format stays the same.
